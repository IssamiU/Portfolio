# Portfólio — Issami Umeoka

Este é o repositório do meu portfólio pessoal, desenvolvido com foco em apresentar meus projetos, habilidades e evolução como desenvolvedor de software.

O site foi construído utilizando React e TypeScript, com uma arquitetura organizada e escalável, priorizando boas práticas de desenvolvimento, componentização e experiência do usuário.

## Principais funcionalidades

- Página inicial com destaque para projetos selecionados
- Página individual para cada projeto com informações detalhadas
- Sistema de ordenação automática na aba de projetos
- Integração com GitHub e GitLab para exibição de repositórios públicos
- Seção de atividade mostrando evolução técnica e estudos
- Layout moderno, responsivo e com animações suaves

## Tecnologias utilizadas

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- TanStack Router

## Objetivo

Criar uma apresentação profissional dos meus trabalhos, demonstrando não apenas os projetos desenvolvidos, mas também minha organização, evolução contínua e forma de trabalhar.
