PROJETOS

Projeto de Aplicação de API - Primeiro Semestre

Descrição: Este projeto foi desenvolvido para coletar e organizar dados disponibilizados pelo site oficial da prefeitura, apresentando informações sobre proposições, frequência, e participação dos vereadores em comissões de forma clara e acessível ao público. Tecnologias Utilizadas: Python, HTML/CSS, JavaScript, MySQL e JIRA. Desafio: Criar uma plataforma interativa que simplificasse a busca e a visualização das informações relacionadas aos vereadores de São José dos Campos. Solução: Desenvolvemos um site utilizando HTML, CSS, Python e um banco de dados MySQL para processar e apresentar os dados, oferecendo uma experiência de pesquisa mais prática e intuitiva para os usuários.
