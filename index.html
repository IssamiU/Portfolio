SOBRE MIM

Sou Issami Umeoka, nascido em 2005, e atualmente estou cursando Desenvolvimento de Software Multiplataforma na Fatec São José dos Campos, com previsão de conclusão em 2027. Minha formação inclui também o Técnico em Informática pelo Senac São José dos Campos, concluído em 2023, onde adquiri sólida base em áreas como programação, desenvolvimento web, engenharia de software e banco de dados. Tenho como objetivo atuar na área da programação que eu possa colocar em prática meus conhecimentos na empresa.

Sou comunicativo, disciplinado e adaptável, com facilidade para trabalhar em equipe e resolver problemas de forma criativa. Minhas experiências em esportes e projetos técnicos fortaleceram minha resiliência e organização, além de me capacitar para aprender rapidamente e colaborar eficazmente.

PROJETOS

API Primeiro Semestre

Descrição: Projeto que organiza e apresenta, de forma interativa e acessível, dados disponibilizados pelo site da prefeitura sobre proposições, frequência, e participação em comissões de cada vereador.

Tecnologias utilizadas: Html, CSS, Flask, Python, Javascript, Tailwind Framework, Railway (BD remoto), Jira, Figma, Microsoft Excel, Visual Studio Code (VSC) e Github.

Atuação: Modelo Ágil - DevTeam - FrontEnd e Backend, raspagem de dados e web design (Html, CSS, Flask, Python, Javascript, Tailwind, Railway)

API Segundo Semestre

Descrição: Desenvolvimento de um Dashboard de Indicadores para a Helpnei, que centraliza e apresenta, de forma interativa, dados sobre lojas, usuários e transações. A interface é responsiva e intuitiva, permitindo o acompanhamento do desempenho da plataforma por meio de métricas como número de lojas criadas, usuários impactados e distribuição geográfica.

Tecnologias utilizadas: React, TypeScript, JavaScript, Tailwind CSS, HTML5, CSS3, Node.js, Express, SQL, Figma e Git.

Atuação: Modelo Ágil - DevTeam - FrontEnd, desenvolvimento de interface, consumo de dados e integração com backend (React, Node.js, Express, SQL, Tailwind).

API Terceiro Semestre

Descrição: Desenvolvimento de uma Plataforma de Gestão de Tarefas e Colaboração para a parceira GSW. O projeto visa otimizar o fluxo de trabalho das equipes, indo além de um simples organizador. A solução integra controle de prioridades, visualização de prazos via calendário, inclusão de anexos para comprovantes e atribuição de responsabilidades em uma interface totalmente responsiva. Além disso, conta com um sistema de autenticação próprio para garantir a segurança e autonomia dos usuários.

Tecnologias utilizadas: React, TypeScript, Java, SpringBoot, Tailwind CSS, HTML5, CSS3, MongoDB, Redis, Figma e Git.

Atuação: Modelo Ágil - Project Owner (P.O) - Documentação de Artefatos, Gerenciamento de Backlog, Planejamento de Sprints, Coordenação de Equipe, Desenvolvimento de WireFrames, Comunicação e ponte direta entre dev-team e cliente, Acompanhamento de Métricas de desempenho (React, TypeScript, Java, SpringBoot, MongoDB, Redis, Tailwind).
